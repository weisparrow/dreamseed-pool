## 数据导出 SOP

每周日 24:00 后由造梦者执行：
1. 登录 DreamField 后台 → 导出本仓库本周种子投放明细 CSV
2. 运行 `scripts/aggregate.py` 聚合到本奖池
3. commit + push，commit message 格式：`pool: 第 N 周更新 (YYYY-MM-DD ~ YYYY-MM-DD)`
4. 月底（每月 1 号）将上月奖池快照上传 IPFS，hash 写入 `audit/`
