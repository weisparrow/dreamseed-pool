# DreamSeed 众筹分奖公示 v1.3

> 本仓库参与 DreamField 主办的 **DreamSeed 种梦计划 — AI 创造者大赛**（2026-05-26 至 2026-11-26，共 180 天）。
> 造梦者承诺：本仓库若获得现金奖励，**90% 的税后净奖金**将以 **DreamField 平台 token（算力券）** 形式按本规则分配给为本仓库投放种子的所有参与者。

## 一、池子结构

| 项目 | 比例 | 形式 |
|---|---|---|
| **每日 token 池** | **税后净奖金 × 90%** | 平均切为 180 份每日池，由 DreamField 官方代充 |
| 运营缓冲（争议/补发） | 3% | 造梦者保管，结束 90 天后未使用部分回填补发 |
| 造梦者本人留存 | 7% | 现金 |

> "税后净奖金" = DreamField 实际打到造梦者账户的金额（已扣 20% 偶然所得税）。
> 示例：仓库获第 1 名（税前 25 万） → 造梦者实到 20 万 → 总池价值 18 万元的 token → **每日池 = 价值 1,000 元的 token**。

## 二、份额计算公式

```
每日池 = (税后净奖金 × 90%) / 180

个人当日所得 = 每日池 × (个人当日有效种子 / 当日全体有效种子)

个人最终 token = Σ(d=1 → 180) 个人当日所得
```

**关键定义**：
- **当日**：以 DreamField 系统时间 00:00:00 至 23:59:59 为一天，每日 24:00 锁定快照。
- **有效种子**：DreamField 后台记录的当日实际投放给本仓库的种子数（解绑提回部分不计入）。
- **参与者**：所有为本仓库投放种子的 DreamField 账号——按同一规则参与每日池分配。

## 三、计算举例

假设最终税后池为 **价值 18 万元 token**，每日池 = **价值 1,000 元 token**。

**场景 A：早期独家支持**
- 第 1 天：张三投 1,000 粒 → 张三独享 1,000 元 token
- 第 2 天：张三 500 粒、李四 500 粒 → 各拿 500 元 token

**场景 B：常态期**
- 第 60 天：5 人各投 1,000 粒 → 各拿 200 元 token

**场景 C：决赛冲刺日**
- 第 175 天：50 人合计投 5 万粒，王五投 1 万粒 → 王五拿 1,000 × (10000/50000) = 200 元 token

> 早期独家支持者投入产出比显著高于后期，体现"种梦"早期支持价值。

## 四、参与门槛

**无门槛。** 当日投放任意数量种子（≥1 粒）即按比例分享当日池。所有参与者使用完全相同的规则。

## 六、参与额外权益（来自 DreamField 平台规则）

除本仓库的众筹分奖外，作为助梦者还可以**直接享受 DreamField 平台原生权益**（详见大赛规则第 10.2 条）：

| 仓库当日种梦榜排名 | 你享受的算力折扣 |
|---|---|
| Top 20% | **6 折** |
| Next 30% | **7 折** |
| Remaining 50% | **8 折** |

**生效规则**（以 DreamField 规则为准）：
- 你为本仓库投出 **至少 1 粒有效种子** 后，从 **次日 00:00** 起即可享受本仓库当日的折扣档。
- 折扣**当日 23:59:59 失效，不可提现，不可叠加使用**。
- 当日已发放的折扣不因解绑或重绑变化。
- 同一时刻只能绑定一个作品，重新绑定后从次日 00:00 起按新作品折扣档生效。

> 也就是说：投种子支持本仓库的同时，你自己用 DreamField 算力本身就能省钱——**这是 DreamField 给所有助梦者的官方权益，不是本众筹方案附加**。

## 七、计量与公示

- **数据源**：种子投放数据全部来自 DreamField 官方后台导出。
- **每日快照**：每日 24:00 后从 DreamField 后台导出当日数据并写入仓库。
- **周更**：每周日更新 [`pool.md`](./pool.md)，列出每位参与者的脱敏 ID + 累计 token 价值。
- **月公证**：每月 1 号将当月奖池快照写入 IPFS 并附时间戳，hash 公示在 `audit/` 目录。

## 八、奖金到账后流程

1. DreamField 公布最终排名 + 资格复核（约决赛后 30 个工作日内）。
2. 奖金到账造梦者账户，造梦者在 **7 个自然日内**整理最终奖池。
3. 造梦者向 DreamField 提交代充 CSV（DreamField UID + 应充 token 数）。
4. DreamField 完成代充后，造梦者公告完成情况 + 凭证截图。

## 九、无奖金情形

若本仓库未进入 Top 32（无现金奖励），**token 池为 0，无分配**。
参与者已享受的折扣权益、星耀榜曝光不予追溯，种子按 DreamField 平台规则处理。

## 十、风险与免责

1. 分配以税后实际到账为准。DreamField 因任何原因未足额打款的，token 池按实际到账比例缩减。
2. **token 性质**：DreamField 平台算力券，不可提现、不可转让、不可兑现金。
3. **token 有效期**：以 DreamField 官方代充时给出的有效期为准。
4. **DreamField 平台规则优先**：本公示与 DreamField 活动规则冲突时，以 DreamField 规则为准。
5. **规则锁定**：仓库报名后，分配规则不再修改（仅可在公示时间窗内修订错别字/澄清歧义）。
6. **未登记 UID 的后果**：参与者若未在公示截止时间前提交 DreamField UID，造梦者保留将其份额转入运营缓冲池的权利。
