# 🌱 DreamSeed Pool · 种梦池

> **DreamSeed 种梦计划 — AI 创造者大赛参赛作品 + 公开众筹分奖项目**

本项目是 [DreamField](https://www.dreamfield.top) 主办的 **DreamSeed 种梦计划 — AI 创造者大赛**（2026-05-26 至 2026-11-26）参赛仓库。

与传统参赛项目不同，**本项目承诺将获奖税后净奖金的 90% 以 DreamField 平台 token（算力券）形式回馈给所有为本仓库投放种子的助梦者**——一个面向参与者的公开众筹分奖实验。

## 💰 众筹分奖机制（核心特色）

| 项目 | 比例 | 形式 |
|---|---|---|
| **助梦者每日 token 池** | **税后净奖金 × 90%** | 平均切为 180 份每日池，DreamField 官方代充 |
| 运营缓冲（争议/补发） | 3% | 造梦者保管，结束 90 天后未使用部分回填 |
| 造梦者本人留存 | 7% | 现金 |

**分配规则**：奖金切成 180 个每日池，**当日按种子投放占比瓜分当日池**。早期支持者享受更高 ROI（独家支持日可独享当日整池）。

## 🌟 如何参与（成为助梦者）

**两步走，缺一不可**：

### Step 1 · 在 DreamField 上支持本仓库
1. 注册/登录 [DreamField](https://www.dreamfield.top)
2. 在大赛活动页搜索本仓库 `dreamseed-pool` 并 **绑定为助梦作品**
3. 使用 DreamField AI 算力 → 自动生产种子 → 自动投放给本仓库

### Step 2 · 邮件登记 DreamField UID（用于后期 token 代充）
> 奖金到账后造梦者需向 DreamField 提交代充名单，**未登记 UID 的参与者无法收到 token**。

📧 **登记邮箱**：[填造梦者邮箱]
📋 **邮件标题**：`DreamSeed 众筹参与登记 - <你的 GitHub 或昵称>`
📝 **邮件需包含**：DreamField 注册账号（脱敏）+ DreamField UID + 公开使用的脱敏昵称 + 联系方式
⏰ **截止时间**：2026-11-25 24:00（建议首次投种子后立即登记）

详见 [`FUNDING.md` 第五节](./FUNDING.md#五如何成为参与者重要)。

## 🎁 你能享受到的两份回报

| 回报 | 触发条件 | 来源 |
|---|---|---|
| **A. 算力使用折扣（即时）** | 投出 ≥1 粒种子，次日 0 点起按本仓库当日榜单档位享受 6/7/8 折 | DreamField 平台官方权益 |
| **B. token 池份额（赛后）** | 累计种子投放 → 按当日占比瓜分每日池 | 本仓库众筹承诺 |

> 投种子越早，B 项投入产出比越高（第 1 天独家支持可独享 1/180 池子）。
> A 项是 DreamField 官方权益，不论本仓库是否最终获奖，每天都生效。
