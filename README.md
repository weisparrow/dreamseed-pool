## 💰 众筹分奖机制（核心特色）

| 项目 | 比例 | 形式 |
|---|---|---|
| **助梦者每日 token 池** | **税后净奖金 × 90%** | 平均切为 180 份每日池，以 DreamField 官方兑换码邮件发放 |
| 运营缓冲（争议/补发） | 3% | 造梦者保管，结束 90 天后未使用部分回填 |
| 造梦者本人留存 | 7% | 现金 |

**分配规则**：奖金切成 180 个每日池，**当日按种子投放占比瓜分当日池**。早期支持者享受更高 ROI（独家支持日可独享当日整池）。

## 🎁 支持本仓库 = 你能拿到两份真实回报

> **不是空头承诺。一份是 DreamField 平台官方现成的折扣，从绑定第二天就开始省钱；另一份是本仓库公开签字画押的奖金分成，180 天每天记账，更新账本。**

### 回报 A：算力立省，每天都生效 💸

绑定本仓库 + 投出 1 粒种子，**次日 0 点起**就开始享受 DreamField 平台折扣。折扣档位由本仓库当日种梦榜排名决定：

| 仓库当日排名 | 你的折扣档 | 等价节省 |
|---|---|---|
| Top 20% | **6 折** | 省 40% |
| Next 30% | **7 折** | 省 30% |
| Remaining 50% | **8 折** | 省 20% |

> 🎯 **本仓库越靠前，你省得越多**——这正是助梦者和造梦者利益绑定的核心。
> 折扣每天结算、每天生效，**和最终获奖与否无关**——这是 DreamField 平台官方权益（详见大赛规则第 10.2 条），我们只是诚实告知你已经拥有它。

### 回报 B：奖金分成，越早投越香 🌱

**本仓库若获奖，税后净奖金的 90% 切成 180 个每日池，按种子投放占比瓜分。**

```
每日池 = (税后净奖金 × 90%) / 180
你当日所得 = 每日池 × (你当日种子 / 全员当日种子)
```

举个直观的例子（假设本仓库拿冠军 → 总池价值 18 万元 token，每日池 1000 元）：

- 第 1 天独家投 1000 粒 → **独享 1000 元 token**
- 第 60 天和 4 人共投 5000 粒 → 拿 200 元 token
- 第 175 天混战日和 49 人共投 5 万粒 → 拿 200 元 token

> ⏰ **第 1 天投的那粒种子，价值是第 175 天的 5 倍以上**——奖励敢第一个上车的人。

### 一句话总结

- **现在投种子** → 明天起每天用 DreamField 6-8 折，立省真金白银
- **持续投种子** → 赛后凭账本凭据按比例拿 token 分成
- **越早投种子** → B 项倍数收益越高，最早期支持者直接独吞当日池
