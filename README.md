# 🌱 DreamSeed Pool · 种梦池

> **DreamSeed 种梦计划 — AI 创造者大赛参赛作品 + 公开众筹分奖项目**

本项目是 [DreamField](https://www.dreamfield.top) 主办的 **DreamSeed 种梦计划 — AI 创造者大赛**（2026-05-26 至 2026-11-26）参赛仓库。

与传统参赛项目不同，**本项目承诺将获奖税后净奖金的 90% 以 DreamField 平台 token（算力券）形式回馈给所有为本仓库投放种子的助梦者**——一个面向参与者的公开众筹分奖实验。

## 💰 众筹分奖机制（核心特色）

| 项目 | 比例 | 形式 |
|---|---|---|
| **助梦者每日 token 池** | **税后净奖金 × 90%** | 平均切为 180 份每日池，DreamField 官方代充 |
| 运营缓冲（争议/补发） | 3% | 造梦者保管，结束 90 天后未使用部分回填 |
| 造梦者本人留存 | 7% | 现金 |

**分配规则**：奖金切成 180 个每日池，**当日按种子投放占比瓜分当日池**。早期支持者享受更高 ROI（独家支持日可独享当日整池）。

## 🌟 如何参与（成为助梦者）

1. 注册/登录 [DreamField](https://www.dreamfield.top)
2. 在大赛活动页搜索本仓库 `dreamseed-pool` 并 **绑定为助梦作品**
3. 使用 DreamField AI 算力 → 自动生产种子 → 自动投放给本仓库
4. 享受当日折扣权益 + 累计 token 池份额

> 投种子越早，投入产出比越高。第 1 天独家支持可独享 1/180 池子。
